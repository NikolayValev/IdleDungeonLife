Idle Dungeon Life

A browser & mobile idle RPG built entirely in TypeScript — pure reducer state, headless simulation, and Playwright test coverage.

▶ Play the Game GitHub Repository

About

Idle Dungeon Life is a fully self-authored idle RPG where players send a procedurally characterized adventurer into increasingly dangerous dungeons, manage resources and equipment, and carry permanent "legacy ash" progress across deaths and rebirths.

The entire game state is managed through a pure, event-sourced reducer — no React, no Vue, no external state library — keeping the core completely decoupled from the Phaser 3 rendering layer and testable in headless Node.js simulations. The project targets both desktop browsers and Android/iOS via Capacitor.

Architecture Highlights

  • Pure Reducer All game state flows through a single reduceGame(state, event) → state function. No direct mutation, no side effects inside the reducer. Every feature is a new event type.
  • Headless Sim A full simulation layer under src/sim/ runs the game engine without Phaser to batch-test balance profiles, generate HTML reports, and verify determinism across seeds.
  • Save Migration Versioned save format (v3) with typed migration helpers that validate and normalise every field on load, ensuring backward-compat across all deployed versions.
  • Seeded RNG All random outcomes (loot, traits, reveals) use a seeded SeededRandomProvider derived from the run seed, making every playthrough fully reproducible.
  • Layered Testing 68 Vitest unit tests cover the reducer, save migration, offline reconciliation, avatar generation, and headless simulation. Playwright e2e specs cover the full run lifecycle in a live browser.

Key Features

Trait System with Evolution

Hidden traits reveal at age, alignment, and dungeon-completion thresholds. Traits evolve mid-run, stacking modifier layers on stats.

Legacy Ash & Prestige

Each death awards ash based on depth, survival time, bosses, and discoveries. Ash unlocks dungeons, jobs, talents, and permanent perks.

Achievement System

Milestone-gated achievements (bosses felled, survival time, paths completed) unlock permanent stat bonuses applied live in every tick.

Three Legacy Paths

Holy, Abyss, and Knowledge paths each unlock a 7-node talent branch and a dedicated perk tree, enabling distinct long-term builds.

50 Items, 16 Traits, 15 Dungeons

Rich authored content with conditional modifiers, alignment gating, tag-based affinity, and rarity-weighted loot tables per dungeon.

Sub-Characters

Up to five independent sub-characters, each with their own meta progression, run state, and automation configuration.

Offline Progression

Up to 24 hours of offline play is reconciled deterministically on load — dungeons complete, traits reveal, and resources accumulate.

Avatar Generator

Procedural SVG character portraits driven by alignment, vitality, equipped item tags, and run seed. Rasterised to PNG atlas for Phaser.
